# Tattoo Shop Management System - Static Demo Website

## Tujuan

Website statis ini digunakan sebagai **demo penawaran jasa pembuatan
website & sistem manajemen Tattoo Shop**, bukan SaaS.

------------------------------------------------------------------------

# Target Audience

-   Tattoo Studio
-   Tattoo Shop
-   Tattoo Artist Independen
-   Studio dengan banyak artist

------------------------------------------------------------------------

# Value Proposition

**Berhenti mengelola booking lewat chat yang berantakan.**

Website membantu: - Booking lebih mudah - Jadwal artist lebih rapi -
Mengurangi no-show - Meningkatkan kepercayaan pelanggan - Operasional
lebih efisien

# Pain Points

-   Booking hanya lewat DM
-   Double booking
-   Jadwal artist sulit dikelola
-   Deposit tidak tercatat
-   Banyak no-show
-   Consent form masih kertas
-   Riwayat pelanggan hilang
-   Sulit menampilkan portfolio artist
-   Tidak ada laporan pendapatan

------------------------------------------------------------------------
